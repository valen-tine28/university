## Style Guidelines
- **Overall aesthetic**: Modern, clean, minimalist, professional
- **Color usage**: Vibrant gradients with indigo → pink → teal flow
- **Design style**: Contemporary tech/SaaS application design
- **Quality**: High resolution, 4K when possible
- **Consistency**: Maintain cohesive style across all images

## Image Dimensions Recommendations
- **Hero Avatar**: 512x512px (circular crop)
- **About Profile**: 800x800px
- **Skill Icons**: 400x400px
- **Project Cards**: 1200x800px (3:2 ratio)
- **Featured Project**: 1600x1200px (landscape)
